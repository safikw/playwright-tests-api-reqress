# Playwright API Tests (JavaScript)

## Setup
1) Install dependencies:
   - `npm install`
2) Set API key:
   - Update `.env` with `API_KEY`

## Run Tests
- Run all tests:
  - `npm run test`
- Run tests with Allure results:
  - `npm run test:allure`

## Allure Report
- Generate and open report:
  - `npm run allure:report`

## Notes
- Allure results are saved in `allure-results/`.
- The HTML report is generated to `allure-report/`.
